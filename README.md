# FPS

A simple FPS in [Zig](https://ziglang.org), based on <https://github.com/lizard-demon/fps>

> [!Note]
> Requires Zig 0.15.2+

## Running

```bash
zig build run
```

## Tests

```bash
zig build test
```

## Controls

Key           | Action
--------------|------------------
**W,A,S,D**   | Move
**Mouse**     | Look around  
**Space**     | Jump (with audio)
**Click**     | Capture mouse
**Escape**    | Release mouse
**Q**         | Quit
